# CompTIA A+ Sprint Prep

Local Flask study app for focused CompTIA A+ review with:

- dark-mode dashboard
- configurable practice quizzes
- flashcards with confidence tracking
- built-in cram sheets, glossary, and domain summaries

## Run

```powershell
python -m pip install -r requirements.txt
python app.py
```

Then open `http://127.0.0.1:5000`.
